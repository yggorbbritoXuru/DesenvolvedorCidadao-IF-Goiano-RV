# DesenvolvedorCidadao-IF-Goiano-RV
Este repositório reúne a implementação de uma Matriz de Avaliação de Impacto para automações RPA em processos de Recrutamento e Seleção, concebida originalmente como artefato de uma dissertação de mestrado profissional em Administração. A aplicação web foi desenhada para ser utilizada por profissionais de RH atuando como citizen developers.
